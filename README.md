## To run Website

1. Open a split terminal 

2. In the first terminal
  - use "cd backend"
  - download requirements with "pip install -r ../requirements.txt"
  - then "uvicorn api:app --reload --host 0.0.0.0 --port 8000"

3. In your second terminal
  - input "npm run dev"
